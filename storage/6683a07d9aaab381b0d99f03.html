• The first CSS library

    The first widely recognized CSS library is Bootstrap, developed by Twitter in 2011. However, before Bootstrap, there were CSS frameworks and tools such as Blueprint CSS (released in 2007) and YUI (Yahoo! User Interface Library), which included a CSS framework component and was released in 2005. These early frameworks helped lay the groundwork for more advanced and popular libraries like Bootstrap.

  • The first JS library

    The first widely recognized JavaScript library is jQuery, which was released in 2006 by John Resig. jQuery significantly simplified client-side scripting of HTML and became extremely popular due to its ease of use and cross-browser compatibility. Before jQuery, there were other JavaScript libraries and frameworks, but none gained as much traction or widespread use. Another notable early library is Prototype, which was released in 2005 and influenced many subsequent JavaScript frameworks.

  • The first React library

    The first library created specifically for use with React is likely React Router. It was released in November 2014 by Ryan Florence and Michael Jackson. React Router is a standard library for routing in React applications, enabling navigation among views of various components, allowing deep linking, and handling location changes.

  • The first Vue library

    The first library created specifically for use with Vue.js is likely Vue Router. Vue Router was developed by Evan You, the creator of Vue.js, and released in 2015 shortly after Vue.js itself, which was released in 2014. Vue Router provides a way to navigate between different views or components in a Vue.js application, making it essential for building single-page applications with Vue.
